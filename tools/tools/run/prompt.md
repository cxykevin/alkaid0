### Tool: `run`

#### Description:

Start a task to run something likes code, command, and so on.

#### Type parameters (which type of task you want to run):

- `"shell"`: Start a system command. 

#### Default shell of `"shell"` type in different OS:

- `"bash"`: Linux.
- `"zsh"`: MacOS.
- `"powershell"`: Windows.
